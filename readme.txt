Descrição do Projeto
Neste projeto, criei uma aplicação web interativa usando HTML, CSS e JavaScript. O foco foi em responsividade e interatividade.

Tecnologias e Funcionalidades
HTML e CSS: Estrutura e estilo responsivo, utilizando flexbox e animações.
JavaScript: Interatividade e efeitos de hover.
Aprendizados
Melhorei minhas habilidades em design responsivo e CSS avançado, além de aprender a adicionar interatividade com JavaScript.
